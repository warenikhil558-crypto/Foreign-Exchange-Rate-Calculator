# Foreign-Exchange-Rate-Calculator

📌 Description

The Foreign Exchange Rate Calculator is a web-based application that allows users to convert currencies in real-time. It uses live exchange rates fetched from an API to provide accurate and up-to-date conversion results.

🎯 Features

💱 Real-time currency conversion

🌍 Supports multiple currencies (INR, USD, EUR, GBP, etc.)

⚡ Fast and user-friendly interface

🔄 Automatic exchange rate updates using API

🛠️ Technologies Used

HTML

CSS

JavaScript

Exchange Rate API

🚀 How to Use

Enter the amount

Select source currency

Select target currency

Click on Convert

Get instant result

📷 Output


Displays converted currency along with updated exchange rate.
